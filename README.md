# ShuttleQueue | TheBreakfastClub Badminton Manager

A professional-grade badminton club management application built with **Next.js 15**, **Tailwind CSS**, **Firebase**, and **Genkit AI**.

## 🚀 Quick Start & Setup

### **1. Firebase Project Connection**
The app is configured to connect to:
- **Project ID**: `studio-8289009920-31c2b`

### **2. Enabling Admin Features**
Security rules prevent client-side elevation to Admin. To enable admin controls for a user:
1. Log into the app to create your account.
2. Go to the **Firebase Console** > **Firestore**.
3. Create a collection named `admin_roles`.
4. Create a document where the **ID** is your **Auth UID**.
5. Add a field `uid` (string) with your UID as the value.

## ✨ Core Features

### **🏟️ Command Center (Dashboard)**
- **Real-Time Monitoring**: Live tracking of active matches and queued players.
- **Admin Hub**: Direct court assignment, manual matchmaking, and session termination.
- **Live Timers**: Visual match duration tracking for every active court.

### **🤖 AI Commissioner (Matchmaking)**
- **Smart Pairing**: Uses Genkit AI and deterministic logic to balance skill levels and variety.
- **Quick Match**: One-click matchmaking that prioritizes waiting players while avoiding repeat partners.
- **Match Analysis**: Explainable AI that provides a 1-sentence justification for every suggested pairing.

### **🏆 Player Experience**
- **Skill Assessment**: 3-point diagnostic tool (Consistency, Movement, Matchplay) to determine skill tier (Level 1-7).
- **Session Gate**: Secure access via 6-digit session codes generated by administrators.
- **Live Leaderboard**: Automated rankings based on wins, win rate, and point differentials.

### **💰 Financials & Payments**
- **Daily Split Calculator**: Admin tool to calculate per-player costs for shuttles, courts, and entry fees.
- **QR Payment Support**: Support for multiple payment methods (e.g., GCash, Maya) with easy QR scanning.
- **Verification Roster**: Real-time tracking of who has settled their daily club dues.
